# NoICANN TLD Registry

Welcome to the NoICANN alternative TLD registry.

## How to Register a TLD

1. Fork this repo.
2. Add your new TLD to `tlds.json` (as a key, value doesn't matter).
3. [Optional] Add metadata to `tlds_meta.json`.
4. Submit a Pull Request.

All TLDs must be:
- Unique
- Lowercase
- 2-63 characters long
